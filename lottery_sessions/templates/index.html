📋 Format attendu

Le fichier CSV doit respecter le format suivant :

  • Extension : .csv
  • Séparateur : ; (point-virgule)
  • Encodage : UTF-8
  • Colonnes obligatoires : nom et prenom
Exemple : 
nom;prenom
Dupont;Jean
Martin;Marie
Bernard;Sophie
Le préfixe "classe_" sera automatiquement ajouté
